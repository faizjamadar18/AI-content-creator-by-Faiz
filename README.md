# Faizbook.ai 🤖📱
### AI-Powered Social Media Platform

Faizbook.ai is a fully-featured social media platform built with **Next.js**, **Clerk Authentication**, and **Convex**.  
It combines modern social networking with AI-powered content creation and editing tools for a seamless, intelligent user experience.

## ✨ Features

- 📸 **Instagram/Facebook-style social media platform**
- 🤖 **AI-powered image editing tools**
- 🪄 **Background removal with 98% precision**
- 🌑 **Professional shadow effects for images**
- ✍️ **Generative AI captions** based on image and title
- ⏱️ **Reduced caption writing time by 90%**
- ❤️ **Like system** for user engagement
- 👥 **Follow / unfollow system**
- 💬 **Real-time comments**
- ⚡ **Scalable interaction engine**
- 🔐 **Secure authentication with Clerk**
